# Information
Pharmanalitik is an online platform to analyse registered pharmatics datas and model them in charts and giving statictical information.  

The Project Uses:
* MySQL  
* HTML5 / CSS3
* Php  
* Javascript (JQuery, AJAX, charts.js, table.js)
